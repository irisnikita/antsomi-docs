**@antscorp/antsomi-hooks** provides a set of hooks that enables you to build your React applications faster

## Install

### Use npm

```bash
npm i @antscorp/antsomi-hooks
```

### Use yarn

```bash
yarn add @antscorp/antsomi-hooks
```
